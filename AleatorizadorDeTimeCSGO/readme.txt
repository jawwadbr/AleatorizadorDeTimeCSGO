NÃO MODIFIQUE NADA DENTRO DOS ARQUIVOS QUE TEM NA PASTA SAVEDNAMES POIS PODE FAZER O PROGRAMA PARAR DE FUNCIONAR!
Caso queira adicionar uma lista diretamente crie um arquivo txt na pasta savedNames e faça igual como vai ser demonstrado a seguir.
Você deve seguir esta ordem
NúmeroDePlayers;Nome0;Nome1;Nome2;...;
Número de players só pode 4 ou 10, caso não for nenhum dos dois a lista não vai carregar e depois do números você coloca o ponto e virgula ';'
Exemplo: 
4;José;Maria;Carlos;Ana;
ESTE README É GERADO JUNTO COM A PASTA SAVEDNAMES ENTÃO SE VOCÊ DELETAR ESSE README ELE SÓ VAI SER GERADO NOVAMENTE CASO A PASTA FOR DELETADA.
